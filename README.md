# Practica-Try-Catch
En esta práctica he trabajado el uso de try-catch en Java para controlar errores comunes al usar programas por consola.
He hecho una calculadora simple, una aplicación bancaria y un sistema de almacenamiento con arrays.

El objetivo principal es evitar fallos cuando el usuario introduce texto en lugar de números, cuando se hacen operaciones no válidas o cuando se intenta acceder a posiciones incorrectas de un array.
